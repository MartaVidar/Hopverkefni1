# Hópverkefni1

## Til að keyra:

	npm install

	npm run dev

## Uppsetning verkefnis:

Verkefninu er skipt upp í þær möppur sem okkur var gefið, undirsíðurnar okkar eru undir möppunni /pages ásamt styles.css og myndirnar sem notaðar voru í verkefninu eru í /img.
